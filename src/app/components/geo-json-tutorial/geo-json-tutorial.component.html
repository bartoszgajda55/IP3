GeoJSON Tutorial This page introduces GeoJSON. What is GeoJSON? GeoJSON is a format used to represent geographic data. A GeoJSON data stucture is a basic JSON object. An object is made of name-value pairs. There are three types of GeoJSON objects: geometry, a feature and a collection of features. GeoJSON syntax The GeoJSON object must have a 'type' value, which is a string that determines the type of the GeoJSON object. It could be either geometry, a feature or a collection of features. Geometry A geometry value describes the type of the GeoJSON object. It could be "Point", "MultiPoint", "Line String", "MultiLineString", "Polygon", "MultiPolygon", "GeometryCollection". Every one of this types except for the GeometryCollection must have a "coordinates" value.
