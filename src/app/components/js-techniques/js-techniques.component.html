Javascript Techniques

Throughout this tutorial page you will recieve information about Javascript Techniques.

There are different techniques that can be used to process and access data and these techniques are defined below:

Creating an Array

The easiest way to encounter this is by identifying/creating (naming the array) and inserting the values inside the array.

Example:

let f = ["String1.1", "String1.2", "String1.3"];

Easiest way to write an IF Statement

A single line if statement. Looking at the example below it declares a ball and depending on the number inserted which is referred to as “x” and checking if it is bigger than 8, then checking if it is true or false.

Example:

let ball = x > 8 ? true : false;

Declaring variables

The example below shows a way on declaring multiple variables in the same line.

Example:

let a, b, c=4;

If Presence

When checking an if statement or also called “if check”. All you have to do is check the variable is ‘true’.

Example:

if (x)

And if it’s not true you could write: if (!x)

A shorthand instead of using charAt()

You can consider this as a shortcut to find a string.

Example

"aString"[0];

Coding Style Part 1 - Using Curly Braces

Curly braces should be not be used on a new line but should be inserted after the key word and there should be a space between the keyword and the opening brace.

For readable code, there should be seperate lines for each statement.

Coding Style Part 2 - Line length and Indents

Line Length

It is suggested that the line length should be around 80-120 characters as it gets hard to read long lines of code.

Indents

There are two types of indents. The first indent is a horiontal indent which is used for two or 4 spaces. The second indent is a vertical indent which is used to divide the code into blocks so that the code is easier to read.

Coding Style Part 3 - Semi Colons & Nesting Levels

Semi Colons

Semi Colons should be used after each statement

Nesting levels

There are two types of indents. The first indent is a horiontal indent which is used for two or 4 spaces. The second indent is a vertical indent which is used to divide the code into blocks so that the code is easier to read.
