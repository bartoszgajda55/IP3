Javascript Techniques

Throughout this tutorial page you will recieve information about Javascript Techniques.

There are different techniques that can be used to process and access data and these techniques are defined below:

Technique 1

Creating an Array

The easiest way to encounter this is by identifying/creating (naming the array) and inserting the values inside the array.

Example:

let f = ["String1.1", "String1.2", "String1.3"];

Technique 2

Easiest way to write an IF Statement

A single line if statement. Looking at the example below it declares a ball and depending on the number inserted which is referred to as “x” and checking if it is bigger than 8, then checking if it is true or false.

Example:

let ball = x > 8 ? true : false;

Technique 3

Declaring variables

The example below shows a way on declaring multiple variables in the same line.

Example:

let a, b, c=4;

Technique 4

If Presence

When checking an if statement or also called “if check”. All you have to do is check the variable is ‘true’.

Example:

if (x)

And if it’s not true you could write: if (!x)
