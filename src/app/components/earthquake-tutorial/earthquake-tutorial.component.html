Earthquake Data Tutorial This page introduces earthquakes, earthquake data fundamentals and where this data can be sourced.

The earthquake page was created to present information about earthquakes for the last hour, all day or all month. For each earthquake a card with information is provided. The card contains data about the magnitude, place, time and type of the earthquake. To gather all of the data we have used the Earthquake Catalog API. Below we have explained how to use the API.

The URL for accessing the data is:

https://earthquake.usgs.gov/fdsnws/event/1/[METHOD[?PARAMETERS]]

Different methods and parameters can be used to filter the data. The parameters are key=value pairs using the HTTP GET method. These methods and parameters are listed below.

Methods

To submit data request:

query

URL: https://earthquake.usgs.gov/fdsnws/event/1/query?format=geojson&starttime=2014-01-01&endtime=2014-01-02

To perform a count on a data request:

count

URL: https://earthquake.usgs.gov/fdsnws/event/1/count?format=geojson

To request available catalogs:

catalogs

URL: https://earthquake.usgs.gov/fdsnws/event/1/catalogs Parameters

Formats: specify the data format

The format can be csv, geojson, kml, quakeml, text or xml. If a format is not specified the default one is quakeml.

URL example: https://earthquake.usgs.gov/fdsnws/event/1/query?format=xml This example will return the specified data in the xml format.

Time: specify the timeframe for which earthquakes happened

All times use the ISO8601 Date/Time format, for example 2019-04-08T18:15:34. The parameters can be endtime, starttime or updatedafter.

URL example: https://earthquake.usgs.gov/fdsnws/event/1/query?starttime=2014-04-01&endtime=2014-04-02 This example will return all earthquake between 1st and 2nd April.
