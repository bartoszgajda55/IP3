Earthquake Data Tutorial This page introduces earthquakes, earthquake data fundamentals and where this data can be sourced.

The earthquake page was created to present information about earthquakes for the last hour, all day or all month. For each earthquake a card with information is provided. The card contains data about the magnitude, place, time and type of the earthquake. To gather all of the data we have used the Earthquake Catalog API. Below we have explained how to use the API.

The URL for accessing the data is:

https://earthquake.usgs.gov/fdsnws/event/1/[METHOD[?PARAMETERS]]

Different methods and parameters can be used to filter the data. The parameters are key=value pairs using the HTTP GET method. These methods and parameters are listed below.

Methods

To submit data request:

query

URL: https://earthquake.usgs.gov/fdsnws/event/1/query?format=geojson&starttime=2019-04-01&endtime=2019-04-02

To perform a count on a data request:

count

URL: https://earthquake.usgs.gov/fdsnws/event/1/count?format=geojson

To request available catalogs:

catalogs

URL: https://earthquake.usgs.gov/fdsnws/event/1/catalogs

 Parameters

Formats: specify the data format

The format can be csv, geojson, kml, quakeml, text or xml. If a format is not specified the default one is quakeml.

URL example: https://earthquake.usgs.gov/fdsnws/event/1/query?format=xml

This example will return the specified data in the xml format.

Time: specify the timeframe for which earthquakes happened

All times use the ISO8601 Date/Time format, for example 2019-04-08T18:15:34. The parameters can be endtime, starttime or updatedafter.

URL example: https://earthquake.usgs.gov/fdsnws/event/1/query?starttime=2014-04-01&endtime=2014-04-02

This example will return all earthquake between 1st and 2nd April.

Location: specify the coordinates for a location

The paramaters can be minlatitude, maxlatitude, minlongitude, maxlongitude. For minlatitude and maxlatitude the value must be between -90 and 90 degrees. For minlongitude and maxlongitude the value must be between -360 and 360 degrees.

URL example: https://earthquake.usgs.gov/fdsnws/event/1/query?format=geojson&minlatitude=42.9&maxlongitude=43

This example will return all earthquakes that happened in the range specified.

To perform a circle search the request should contain all of the following parameters: latitude (value between -90 and 90), longitude (value between -180 and 180), maxradius (value between 0 and 180).

URL example: https://earthquake.usgs.gov/fdsnws/event/1/query?format=geojson&latitude=55.864239&longitude=-4.251806&maxradius=20

This example will return all earthquakes for the specified coordinates.

Other parameters

Some other useful parameter is orderby, which orders the request by time, time-asc, magnitude, magnitude-asc.

URL example: https://earthquake.usgs.gov/fdsnws/event/1/query?format=geojson&orderby=time

This example will return all earthquakes ordered by time in descending order.

Another parameter is maxmagnitude which limits the earthquakes with a magnitude smaller than the specified magnitude.

URL example: https://earthquake.usgs.gov/fdsnws/event/1/query?format=geojson&maxmagnitude=8

This example will return earthquakes with magnitude less than the value specified.
