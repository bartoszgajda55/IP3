Visualizations

About: This page contains 14 different visualizations, displayed on the Google Map in the form of tiles. You can select every of them, using the dropdown below, which will plot the appropriate tiles over the map, along with presenting brief information about the current layer displayed. All of these different visualizations are displayed in one page only as opposed to four specified to project specification. This is beacuse one page requires less resources to display, therefore application is smaller and faster.

Technologies used: Google Map, OpenWeatherMap API, Google Map Image Map API

Development: A Google Map object, is split into tiles, each representing separate part of the globe. After selecting a layerr, each tile is fetched from OpenWeatherMap API, using 4 variables: Zoom of the Map, Layer Code, X coordinates and Y coordinates. Zomming in our out triggers same functions that fetches the tiles for appropriate zoom level automatically.

Result: User can get valuable and interesting information about many weather all around the globe. The data presented in the visualization in innovative way is easy to understand and fun to interact with. The User can gain more memorable information and compare different layers across continents and countries.
